PRAKTIKUM SISTEM INFORMASI GEOGRAFIS BERBASIS WEB

Ujian Tengah Semester

Praktikum Sistem Informasi Geografis Berbasis Web merupakan mata kuliah pilihan yang ditawarkan oleh Teknik Geodesi UGM. Mata kuliah ini awalnya dibuka pada semester ganjil, namun khusus tahun pelajaran ini dibuka di semester genap. Rangkaian praktikum dimulai dari pengenalan HTML secara umum yang dijelaskan pada minggu kedua. Minggu ketiga dan keempat membahas tentang CSS dan JavaScript. Pada minggu kelima, menurut saya ini paling penting karena sudah masuk ke materi GitHub untuk melakukan pengunggahan tugas. Penutup praktikum sebelum UTS, pada minggu keenam dan minggu terakhir dibahas mengenai pembuatan peta menggunakan LeafletJS.
